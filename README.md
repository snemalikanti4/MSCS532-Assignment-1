âš™ï¸ How the Algorithm Works

Insertion Sort builds the final sorted list one element at a time.
In this implementation:

Each element is compared with those before it

Elements smaller than the current value are shifted right

The current value is inserted into its correct position

The comparison logic is reversed to ensure decreasing order
